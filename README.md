# DJO Dictee
Repository of the DJO Dictee, used to test the spelling capabilities of members of DJO.

## Author
The DJO Dictee is proudly made by [ngkon](https://github.com/ngkon6).

## How it's made
The DJO Dictee is made using [Bun](https://bun.sh), with [Hono](https://hono.dev) for the web server and [Socket.IO](https://socket.io) for fancy real-time updating stuff.
